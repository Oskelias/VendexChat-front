# Vendechat Front

Public storefront for Vendechat.

Customer-facing website where users:
- browse products
- add to cart
- checkout
- send orders directly to WhatsApp

## Stack
- Next.js
- TypeScript
- Tailwind
- Cloudflare Pages

## Local development

npm install
npm run dev

## Build

npm run build

## Deploy
Auto-deployed via Cloudflare Pages connected to GitHub.

## Env variables

NEXT_PUBLIC_API_URL=https://api.vendechat.app
